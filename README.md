# retoFinal
## Objetivo: 
Un repositorio del equipo con ramas por integrante y al menos 2 Pull Requests. Usen la plantilla de entregable. 

    Definan un repositorio del equipo (uno solo) y abran un Codespace.
    Asignen roles: Driver/Navigator/Tester/Documentador (rotación rápida).
    Cada integrante crea una rama: feat/nombre-navbar, docs/nombre-readme, etc.
    Cada rama debe tener 1–2 commits pequeños con mensajes claros.
    Abren PRs, hacen al menos 1 comentario de revisión y hacen merge.
    Registra links: repo + PR(s) + evidencia (capturas o enlaces).
    Reflexión: ¿cómo usarías ramas/PR como evidencia y para enseñar colaboración?
